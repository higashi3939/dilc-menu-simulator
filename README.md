# DILC Menu Simulator

デジタル一眼カメラ（DILC: Digital Interchangeable-Lens Camera）の液晶メニュー画面をブラウザ上でシミュレーションするツールです。  
UIプロトタイピングや仕様検討に使用することを目的にしています。

本ツールはジョークソフトです。

![Version](https://img.shields.io/badge/version-8.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

---

## 特徴

- **単一HTMLファイル** — 外部依存なし。ダウンロードしてブラウザで開くだけで動作
- **静止画 / 動画モード** — モード切替でタブ構成が完全に切り替わる
- **実機準拠のタブアイコン** — 単色シルエットで実機の見た目を再現
- **日本語 / 英語 対応** — タブ名・項目名・選択肢すべてを即時切替
- **実機準拠のメニュー定義** — リファレンス機種（Canon EOS R5 Mark II）の公式マニュアル（cam.start.canon）をもとにした全メニュー項目
- **動画記録サイズウィザード** — 解像度・フレームレート・圧縮方式をアイコングリッドで設定
- **メイン記録形式と排他制御** — 実機マニュアル準拠の組み合わせ制限を反映（無効セルは灰色化、自動補正あり）
- **タイムラプス動画ウィザード** — インターバル・コマ数・撮影所要時間のリアルタイム計算
- **スクリーンショット機能** — 液晶画面の表示をワンクリックでPNGとして保存（外部ライブラリ不要）
- **JSONエディター** — メニュー定義をその場で編集・即時反映
- **ナビゲーションログ** — 操作履歴をリアルタイムで記録

## 使い方

### 基本操作

1. `menu-simulator-v8.html` をブラウザで開く
2. 画面上部のモードスイッチで **📷 静止画** / **🎬 動画** を切替
3. コントロールパッドまたはキーボードでメニューを操作

### キーボードショートカット

| キー | 動作 |
|---|---|
| `←` / `→` | タブ・ページ移動（端でラップアラウンド） |
| `↑` / `↓` | 項目移動 |
| `Enter` | 決定（SET） |
| `Esc` / `BackSpace` | 戻る（MENU） |
| `M` | 静止画 / 動画 モード切替 |
| `L` | 日本語 / English 言語切替 |

### メニュー定義のカスタマイズ

右側の **JSONエディター** でメニュー構成を編集し、「▶ 適用」で即時反映できます。

```json
{
  "stills": {
    "tabs": [ ... ]
  },
  "movie": {
    "tabs": [ ... ]
  }
}
```

## 専用ウィザード

### 動画記録サイズ

動画モード → MOVIE タブ → Page 1 → `動画記録サイズ` で起動。

- **3タブ構成**：解像度（12種）/ フレームレート（解像度依存）/ 圧縮方式（解像度依存）
- **タブ行は現在選択値のSVGアイコン**で表示（テキストラベルではない）
- **実機準拠のアイコンデザイン**：
  - 解像度＝フィルム枠デザイン + 解像度名（`Fine` 版は2段組）
  - フレームレート＝白地黒文字の角丸ピル型 `XX.XXP`
  - 圧縮方式＝白ラベル（`Intra` / `LGOP` / `RAW`）+ 修飾（標準/高画質↑/軽量↓）
- **排他制御**：
  - 解像度ごとに利用可能な fps / 圧縮方式が変わる（`MR_FPS` / `MR_COMP`）
  - **メイン記録形式（`m1_3`）と連動**：RAW専用解像度、8K対応のHEVC S系、Intra系専用のAVC422 10bit など、実機マニュアル準拠の組み合わせ制限を反映
  - 無効な組み合わせは灰色化表示、`Enter`/`SET` は拒否
  - メイン記録形式を変更すると現在値が自動補正される
- **初期値**：Full HD / 29.97P / 標準 Long GOP
- 確定するとメニュー行にSVGアイコン3連で設定値を表示。再度開くと前回選択値からスタート

### タイムラプス動画

動画モード → MOVIE タブ → Page 6 → `タイムラプス動画` で起動。

- 撮影間隔（時:分:秒）・撮影枚数・動画記録サイズほか8項目を設定
- 動画記録サイズはメイン側の `m1_1` と同じくSVGアイコン3連（解像度＋fps＋圧縮方式）で表示
- ヘッダータイトルが現在の階層（カーソル行のラベル）に動的に切り替わる
- 撮影所要時間・再生時間（29.97fps換算）をリアルタイム計算・表示
- メニュー行に **する / しない** が表示され、ウィザード内のタイムラプス撮影設定とリアルタイム連動

## スクリーンショット保存

コントロールパッドの右側にある **📸 保存** ボタンで、液晶画面の現在の表示をPNGファイルとしてダウンロードできます。

- 出力サイズ：520×346px（高DPI環境では `devicePixelRatio` を考慮した倍率で出力）
- ファイル名：`dilc-menu-YYYYMMDD-HHMMSS.png`
- 外部ライブラリ不要（ブラウザ標準の SVG `<foreignObject>` + Canvas API で実装）
- メイン画面・ウィザード（動画記録サイズ・タイムラプス動画）・ダイアログ表示中も含めて、現在表示されている液晶画面そのまま保存可能

| セクション | 内容 |

| S6: WIZARD — MOVIE REC SIZE | 動画記録サイズウィザード |
| S7: WIZARD — TIMELAPSE | タイムラプス動画ウィザード |

## 変更履歴

### v8

#### プロジェクト名の汎用化（公開準備）

- 「EOS R5 Mark II Menu Simulator」→「DILC Menu Simulator」に改名
- リファレンス実装としては引き続き Canon EOS R5 Mark II のメニュー構成を採用
- スクリーンショット保存ファイル名: `eos-r5m2-menu-...` → `dilc-menu-...`

#### スクリーンショット機能の追加

- コントロールパッド右側に📸スクリーンショットボタンを設置
- 液晶画面（520×346px）をワンクリックでPNGとして保存
- 外部ライブラリ不要（SVG `<foreignObject>` + Canvas API で実装）
- 高DPI対応、自動タイムスタンプ付きファイル名
- Chrome の tainted canvas エラーを data URL 方式で回避

### v7

#### タイムラプス動画ウィザード

- ラベル整理（実機準拠）：「タイムラプス」→「タイムラプス撮影」、「インターバル」→「撮影間隔」、「撮影コマ数」→「撮影枚数」、「画面自動オフ」→「モニター自動消灯」、「撮影ごとのビープ音」→「撮影時の電子音」
- 配色を全面刷新（青系 → 白＋親タブ色）
- 選択行スタイルをメイン側と統一（上下1px・左右10pxの親タブ色ボーダー）
- 動画記録サイズの設定値をSVGアイコン3連で表示（メイン側 `m1_1` と同じ仕様）。動画記録サイズ選択UIも6行のアイコン3連リストに刷新（ラジオ・テキストなし）
- ヘッダータイトルの動的切り替え（編集中はカーソル行のラベル＝階層を表示、各編集UI内の独自タイトルバーは削除）

#### 動画モード Page 6 メニュー項目

- `m6_1` プリ記録設定 → プレ記録設定。`type:'select'` 化、選択肢「切 / 入」、初期値「切」
- `m6_2` タイムラプス動画は `type:'action'` のまま、メニュー行に **する / しない** が表示されるように（実機準拠）。ウィザード内のタイムラプス撮影設定とリアルタイム連動

#### メニュー項目行の50:50レイアウト

- 行幅を左右50%ずつに分割し、設定値は行の中央位置から左寄せで表示
- メイン側のすべての行・アイコン3連行・タイムラプスウィザード内の行で統一

#### `ivalue()` の汎用化

- `options` の有無を問わず、value に `/` が含まれれば言語別に分割
- `type:'action'` の項目でも言語ペア形式の value で状態表示が可能に


### v6

#### 動画モード Page 1 メニュー項目

- ラベル変更：「高フレームレート」→「ハイフレームレート」、「主記録形式」→「メイン記録形式」、「デュアル撮影（静止画と動画）」→「動画撮影中静止画記録」
- 選択肢変更：ハイフレームレート「切/入」、動画撮影中静止画記録「OFF/ON」（実機表記）
- メイン記録形式の選択肢を5種類の技術名（RAW / XF-HEVC S YCC422 10bit / XF-HEVC S YCC420 10bit / XF-AVC S YCC422 10bit / XF-AVC S YCC420 8bit）に拡張

#### メイン記録形式 ↔ 動画記録サイズの排他制御

- 実機マニュアル準拠の排他関係を実装（[UG-03_Shooting-2_0040.html](https://cam.start.canon/ja/C017/manual/html/UG-03_Shooting-2_0040.html)）
- メイン記録形式に応じて選べる解像度・圧縮方式が変わる（RAW専用、8K対応HEVC S系、Intra系専用AVC422 10bit など）
- 無効セルは灰色化、Enter/SET 拒否、ナビゲーションログに `✗ DENY` を出力
- メイン記録形式変更時に現在値が無効化された場合は自動補正

- メインタブアイコンを実機の単色シルエットに刷新

- 形状更新：AF（テキストのみ）/ Play（▶矩形）/ Comm（波線）/ Ctrl（円グラフ風）/ C.Fn（カメラ背面ビュー）

- 圧縮方式アイコンをラベル＋修飾の3パターンに刷新（標準 / 高画質↑ / 軽量↓）
- RAW・軽量RAWの専用アイコンを追加
- タブ行をテキスト → 現在選択値のSVGアイコン3個に変更
- 初期値を Full HD / 29.97P / LGOP に変更
- ウィザード起動時に前回選択値から状態復元
